一、查看操作系统信息

1、uname -a 查看操作系统信息

$ uname -a
Linux VM_101_50_centos 2.6.32-642.6.2.el6.x86_64 #1 SMP Wed Oct 26 06:52:09 UTC 2016 x86_64 x86_64 x86_64 GNU/Linux

2、cat /etc/issue 查看内核

$ cat /etc/issue
CentOS release 6.3 (Final)
Kernel \r on an \m

3、cat /etc/redhat-release # 查看内核

$ cat /etc/redhat-release 
CentOS release 6.3 (Final)

4、其它

hostname   # 查看计算机名 
env    # 查看环境变量
